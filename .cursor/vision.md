# Opportunity.ai Vision - Cursor Reference

**Purpose:** Concise vision reference optimized for Cursor context and @ references.

## Mission

AI-assisted job search and application management platform for tech professionals conducting multi-country job searches.

## Core Value Propositions

1. **AI-Powered Job Discovery**
   - Multi-country/platform aggregation
   - Semantic search and matching
   - Personalized fit score ranking

2. **Tailored Application Materials**
   - CV generation tailored to job descriptions
   - Cover letter generation
   - **Outreach Draft Generator** (email/LinkedIn messages)

3. **Job Application Tracking**
   - Personal ATS/CRM pipeline tracker
   - Status stages: Saved, Applied, Interview, Offer, Rejected
   - Follow-up reminders

4. **Personalized Insights**
   - Resume/profile scoring
   - Skill gap analysis
   - Learning recommendations

5. **User Control & Privacy**
   - GDPR compliant
   - Privacy-by-design
   - Full data control (export/delete)

## Target Users

Tech professionals (engineers, PMs, designers, data scientists) seeking international or remote opportunities.

## Competitive Context

- **Teal**: Chrome extension, basic tracking
- **AIApply.com**: Automated resume/cover letter
- **Jobscan/ResyMatch**: Resume-to-job matching only
- **Opportunity.ai Differentiator**: Unified platform with discovery + generation + tracking

## Key Principles

- **Compliance First**: No scraping, official APIs only, respect ToS
- **Human-in-the-Loop**: All AI outputs require user approval
- **Privacy-by-Design**: GDPR compliant from day one
- **User Control**: Transparent AI, full data control

## Development Approach

- **UI-First**: Build UI prototype first, then integrate backend
- **Clean Architecture**: Progressive enhancement (presentation → service → domain → data)
- **Incremental**: Small commits, continuous verification
